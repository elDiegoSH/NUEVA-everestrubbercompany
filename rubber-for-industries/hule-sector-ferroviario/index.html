Hule Especializado para Industria Ferroviaria

En el sector ferroviario, los materiales deben soportar cargas extremas, vibraciones constantes y condiciones climáticas variables. Los productos de hule EVEREST® están diseñados para aplicaciones ferroviarias críticas.

Resistencia a Cargas Dinámicas

Capacidad para soportar cargas extremas y movimientos en sistemas ferroviarios.

Amortiguación de Vibraciones

Reducción de vibraciones y ruido en vagones, locomotoras y sistemas de vía.

Aplicaciones en Industria Ferroviaria

Soluciones específicas para sistemas de transporte ferroviario

Placas de Asiento para Rieles

Placas de hule para aislamiento y amortiguación entre rieles y durmientes.

  • Reducción de vibraciones
  • Aislamiento eléctrico
  • Protección contra corrosión

Amortiguadores para Vagones

Sistemas de amortiguación para bogies, suspensiones y sistemas de choque.

  • Absorción de impactos
  • Estabilidad dinámica
  • Larga duración

Juntas de Dilatación

Juntas de hule para puentes ferroviarios y estructuras que permiten movimiento.

  • Absorción de movimientos
  • Resistencia a compresión
  • Estabilidad dimensional

Sellos para Puertas

Sistemas de sellado para puertas de vagones que garantizan estanqueidad.

  • Estanqueidad total
  • Resistencia UV y ozono
  • Flexibilidad permanente

Pisos para Vagones

Pisos de hule para vagones de pasajeros y carga con propiedades antiderrapantes.

  • Antiderrapante
  • Resistencia al desgaste
  • Confort acústico

Componentes para Sistemas

Componentes de hule para sistemas de frenado, acoplamiento y control.

  • Precisión dimensional
  • Resistencia a fatiga
  • Compatibilidad con sistemas
